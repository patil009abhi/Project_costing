A modern **Project Costing Application** built using React, Node.js, and MySQL.  
This application helps manage and estimate project costing including cement, steel, and labor expenses with complete history tracking.

---

# 🏗️ Architecture

| Layer | Technology | Purpose |
|--------|------------|----------|
| Frontend | ⚛️ React | User Interface |
| Backend | 🟢 Node.js | Business Logic & APIs |
| Database | 🐬 MySQL | Cost & History Storage |

---

# ✨ Features

✅ Client-wise Project Costing  
✅ Cement Cost Estimation  
✅ Steel Cost Estimation  
✅ Labor Cost Calculation  
✅ Cost History Tracking  
✅ React Based UI  
✅ Node.js Backend APIs  
✅ MySQL Database Storage  
✅ Docker Containerization  
✅ GitHub Actions CI/CD Automation  
✅ Self Hosted Runner Deployment  

---

# ⚙️ Application Workflow

```text
React Frontend
       ↓
Node.js Backend API
       ↓
MySQL Database
       ↓
Project Cost & History Results
```

---

# 📦 Frontend

The frontend UI allows users to:

- Enter client name
- Add cement cost
- Add steel cost
- Add labor cost
- View project costing history

---

# 🛠️ Backend

The backend handles:

- API requests
- Cost calculations
- Database operations
- History management
- Business logic processing

---

# 🐬 Database

All project costing data including:

- Client Name
- Cement Cost
- Steel Cost
- Labor Cost
- Total Estimated Cost
- History Records

are stored securely in MySQL database.

---

# 📊 History Tracking

The application provides a history section on the UI to track:

- Previous estimations
- Client records
- Cost updates
- Historical project data
